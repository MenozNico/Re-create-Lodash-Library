# Re-create-Lodash-Library
### *Capstone project for Programming in JavaScript* ###

In this project I will be implementing *ten methods* from the incredibly popular Lodash JavaScript library:
1. **clamp()**
1. **inRange()**
1. **words()**
1. **pad()**
1. **has()**
1. **invert()**
1. **findKey()**
1. **drop()**
1. **dropWhile()**
1. **chunk()**
